{{< lead >}}
قسم مخصص لـ **حزم ومكتبات PHP العملية فقط** — هتتعلم فيه إزاي تستخدم الحزم الجاهزة علشان تختصر وقت التطوير وتبني مشاريع قوية وقابلة للتوسّع.
{{< /lead >}}

في هذا القسم ستتعلم:
- كيفية استخدام Composer لإدارة الحزم  
- أفضل الممارسات في تركيب وتحديث الحزم  
- طريقة دمج الحزم داخل مشاريعك بشكل احترافي  
- فهم مفهوم Autoloading وكيفية تنظيم الأكواد  
- بناء مشاريع صغيرة ومتوسطة تعتمد على الحزم  

**حزم ومشاريع مقترحة داخل القسم:**
- استخدام **GuzzleHTTP** لعمل HTTP Requests  
- إنشاء API بسيط باستخدام **Slim Framework**  
- التعامل مع قواعد البيانات باستخدام **Eloquent ORM** بدون Laravel  
- إنشاء نظام تسجيل دخول باستخدام **PHP JWT**  
- استخدام **Monolog** لكتابة سجلات احترافية  
- قراءة ومعالجة الملفات باستخدام **League Flysystem**
