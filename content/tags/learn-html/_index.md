---

{{< lead >}}
دليل شامل لتعلم **HTML فقط** — من الوسوم الأساسية إلى كيفية بناء صفحات ويب منظمة وقابلة للفهم من المتصفحات ومحركات البحث.
{{< /lead >}}

في هذا الوسم ستجد:
- شروحات تفصيلية للوسوم الأساسية مثل (`<div>`, `<header>`, `<nav>`, `<section>`, `<article>`، وغيرها).  
- شرح بنية صفحة HTML من البداية للنهاية، وفهم دور كل وسم.  
- كيفية كتابة كود HTML نظيف وسهل القراءة.  
- نصائح للوصولية (Accessibility) داخل HTML فقط، بدون دخول في CSS أو JavaScript.  
- إنشاء عناصر HTML مثل: النماذج (Forms)، الجداول، القوائم، الروابط، الصور، والفيديو.  

**مواضيع مقترحة للقراءة:**
- مقدمة سريعة إلى HTML وبنية المستند  
- شرح وسم `<head>` وما يحتويه  
- كيفية إنشاء نماذج احترافية باستخدام HTML فقط  
- أفضل الممارسات لكتابة HTML منظم وقابل للتطوير
