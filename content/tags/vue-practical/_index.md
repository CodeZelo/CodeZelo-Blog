{{< lead >}}
قسم مخصص للتطبيقات العملية باستخدام **Vue.js** — من بناء مكونات بسيطة إلى تطوير تطبيقات كاملة تعتمد على Vue Router و Pinia.
{{< /lead >}}

في هذا القسم ستتعلم:
- إنشاء مكونات (Components) قابلة لإعادة الاستخدام  
- التعامل مع البيانات والـ Props والأحداث  
- استخدام Computed وWatch لكتابة منطق أذكى  
- ربط Vue بالـ API وجلب البيانات  
- بناء صفحات متعددة باستخدام Vue Router  
- إدارة الحالة باستخدام Pinia أو Vuex  
- مشاريع صغيرة ومتوسطة لبناء خبرة قوية  

**مشاريع مقترحة داخل القسم:**
- إنشاء To-Do App باستخدام Vue  
- معرض صور ديناميكي عبر API  
- بناء نظام Tabs Component  
- إنشاء صفحة تسجيل دخول بواجهة كاملة  
- لوحة تحكم Dashboard بسيطة  
- تطبيق CRUD كامل باستخدام Vue + API خارجي
