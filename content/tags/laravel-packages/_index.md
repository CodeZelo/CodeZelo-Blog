{{< lead >}}
قسم مخصص لـ **حزم Laravel العملية فقط** — هتتعلم فيه إزاي تستفيد من الإيكوسيستم العملاق لإطار Laravel وتضيف وظائف قوية لمشاريعك بدون مجهود كبير.
{{< /lead >}}

في هذا القسم ستتعلم:
- كيفية تثبيت الحزم باستخدام Composer  
- أفضل الطرق لدمج الحزم مع خدمات Laravel  
- التعامل مع Service Providers و Facades  
- كتابة كود قابل للتطوير بالاعتماد على حزم جاهزة  
- بناء مشاريع تعتمد بالكامل على أشهر الحزم  

**حزم ومشاريع مقترحة داخل القسم:**
- إنشاء نظام تسجيل دخول كامل باستخدام **Laravel Breeze** أو **Jetstream**  
- إدارة الصلاحيات والأدوار باستخدام **Spatie Laravel Permission**  
- رفع وتحويل الصور بواسطة **Spatie Media Library**  
- إنشاء API قوي باستخدام **Laravel Sanctum** أو **Passport**  
- جدولة المهام والعمليات Background Jobs باستخدام **Laravel Horizon**  
- بناء لوحة تحكم كاملة باستخدام **FilamentPHP** أو **Nova**
