{{< lead >}}
قسم مخصص لتطبيقات **Backend** — من مشاريع CRUD بسيطة إلى APIs متكاملة، لتقوية خبرتك في البرمجة الخلفية وإدارة البيانات.
{{< /lead >}}

في هذا القسم ستتعلم:
- إنشاء CRUD كامل باستخدام PHP أو Laravel  
- تصميم REST APIs وربطها بقواعد البيانات  
- التحقق من البيانات وإدارة المستخدمين  
- التعامل مع الملفات والرفع والتحكم بها  
- إدارة قواعد البيانات باستخدام MySQL  
- كتابة كود نظيف وقابل للتطوير  

**مشاريع مقترحة داخل القسم:**
- CRUD كامل لإدارة المنتجات أو المقالات  
- نظام تسجيل دخول وتسجيل مستخدمين  
- REST API بسيط لتطبيق To-Do  
- لوحة تحكم Dashboard للمستخدمين  
- رفع ملفات ومعالجتها  
- إرسال إيميلات وإشعارات
