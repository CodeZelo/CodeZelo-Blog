{{< lead >}}
قسم مخصص للتطبيقات العملية في **Laravel** — هنشتغل على مشاريع واقعية، نستخدم فيها الـ Eloquent، الـ Routing، الـ Validation، الـ Middleware، وبناء APIs كاملة.
{{< /lead >}}

في هذا القسم ستتعلم:
- بناء CRUD كامل باستخدام Laravel  
- التعامل مع النماذج (Forms) والتحقق من البيانات  
- استخدام Eloquent ORM بشكل عملي  
- العمل مع قواعد البيانات (MySQL) داخل مشاريع واقعية  
- بناء REST API كاملة  
- نظام تسجيل دخول وتسجيل مستخدمين  
- رفع الملفات والصور ومعالجتها  
- إرسال الإشعارات والإيميلات  
- مشاريع صغيرة ومتوسطة تقوّي خبرتك

**مشاريع مقترحة داخل القسم:**
- إنشاء CRUD بسيط للأقسام أو المنتجات  
- بناء API لمهمات To-Do  
- نظام تسجيل دخول كامل مع Auth  
- إنشاء Blog بسيط (مقالات + تصنيفات)  
- رفع الصور مع التحقق والتخزين  
- Dashboard صغيرة لإدارة المستخدمين  
- إرسال إيميل تأكيد عند إنشاء مستخدم
