{{< lead >}}
قسم مخصص لتطبيقات **Fullstack** — تعلم بناء تطبيقات ويب متكاملة من الواجهة الأمامية إلى الخلفية خطوة بخطوة.
{{< /lead >}}

في هذا القسم ستتعلم:
- دمج واجهة Frontend مع Backend  
- بناء CRUD كامل مع ربط قاعدة البيانات  
- استخدام Vue.js أو JavaScript للتفاعل في الواجهة  
- تصميم REST API وربطها بالواجهة الأمامية  
- إدارة المستخدمين، الجلسات، والتحقق من البيانات  
- بناء مشاريع حقيقية قابلة للتطوير  

**مشاريع مقترحة داخل القسم:**
- تطبيق To-Do كامل (Frontend + Backend)  
- مدونة بسيطة مع نظام CRUD للمقالات  
- لوحة تحكم Dashboard متكاملة  
- تطبيق تسجيل دخول + إدارة مستخدمين  
- معرض صور ديناميكي مرتبط بقاعدة بيانات  
- مشروع متجر إلكتروني صغير
