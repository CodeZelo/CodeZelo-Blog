قسم مخصص للتطبيقات العملية باستخدام **HTML + CSS** فقط — بدون JavaScript.  
هنا هتلاقي مشاريع جاهزة، أمثلة قابلة للتطوير، وتمارين تخليك تبني واجهات حقيقية من البداية للنهاية.
{{< /lead >}}

في هذا القسم ستتعلم:
- إنشاء صفحات هبوط (Landing Pages) كاملة باستخدام HTML وCSS.  
- تصميم نماذج احترافية مع تنسيقات جذابة.  
- بناء ترويسة (Header) مع قائمة تنقل متجاوبة.  
- تنسيق جداول، بطاقات (Cards)، أقسام خدمات وشهادات العملاء.  
- تحويل أكواد HTML خام إلى تصميم كامل باستخدام Flexbox وGrid.  
- مشاريع صغيرة إلى متوسطة لتقوية مهاراتك.

**مشاريع مقترحة داخل القسم:**
- صفحة هبوط بسيطة لمتجر إلكتروني  
- صفحة “About Us” احترافية  
- تصميم نموذج تواصل + تنسيقات CSS  
- معرض صور (Gallery) باستخدام Grid  
- صفحة تسجيل دخول كاملة (Login Page)  
- إعادة تصميم صفحة بورتفوليو شخصية
