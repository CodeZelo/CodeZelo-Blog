{{< lead >}}
ابدأ رحلتك في تعلم **PHP** خطوة بخطوة — اللغة الأساسية لبناء المواقع الديناميكية، من المفاهيم الأساسية حتى كتابة سكربتات متكاملة.
{{< /lead >}}

في هذا القسم ستتعلم:
- مقدمة إلى PHP وكيف تعمل على السيرفر  
- المتغيرات، الأنواع، العمليات، والتحكم في التدفق  
- الدوال وكيفية تنظيم الكود  
- التعامل مع الملفات والـ Sessions والـ Cookies  
- ربط PHP مع قواعد البيانات (MySQL)  
- أفضل الممارسات لكتابة كود نظيف وآمن  

**مواضيع مقترحة:**
- تشغيل PHP لأول مرة  
- فهم الـ Server Side Processing  
- كيفية استخدام المتغيرات والمصفوفات  
- إنشاء صفحات ديناميكية باستخدام PHP  
- أساسيات التعامل مع قواعد البيانات
