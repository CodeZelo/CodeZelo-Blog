{{< lead >}}
ابدأ رحلة تعلم **Vue.js** — واحد من أسهل وأقوى أُطر JavaScript لبناء واجهات أمامية تفاعلية وسريعة.
{{< /lead >}}

في هذا القسم ستتعلم:
- مقدمة إلى Vue.js ولماذا هو من أسهل الأُطر  
- كيفية إنشاء مشروع Vue من البداية  
- فهم مبدأ المكونات (Components)  
- التعامل مع البيانات عبر Data وProps  
- الأحداث (Events) والتفاعل بين المكونات  
- التوجيه (Vue Router)  
- إدارة الحالة باستخدام Pinia أو Vuex  

**مواضيع مقترحة:**
- ما هو Vue ولماذا يُفضل على React للمبتدئين  
- أول تطبيق لك باستخدام Vue  
- فهم نظام المكونات خطوة بخطوة  
- كيفية تمرير البيانات بين المكونات  
- التعامل مع النماذج في Vue  
- بناء تطبيق بسيط باستخدام Vue Router
