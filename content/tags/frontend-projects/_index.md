{{< lead >}}
قسم مخصص لتطبيقات **الواجهة الأمامية (Frontend)** — من مشاريع صغيرة إلى متوسطة لتقوية مهاراتك في تصميم وبناء واجهات تفاعلية.
{{< /lead >}}

في هذا القسم ستتعلم:
- بناء صفحات ويب متكاملة باستخدام HTML وCSS  
- إضافة التفاعلية باستخدام JavaScript  
- بناء مكونات واجهة باستخدام Vue.js  
- استخدام Flexbox و Grid لتصميم متجاوب  
- مشاريع جاهزة لتقوية خبرتك العملية  
- تحسين تجربة المستخدم والتجاوب مع مختلف الأجهزة  

**مشاريع مقترحة داخل القسم:**
- صفحة هبوط (Landing Page) تفاعلية  
- معرض صور ديناميكي  
- نموذج تواصل متكامل  
- لوحة بطاقات (Cards Layout)  
- سلايدر صور تفاعلي  
- Portfolio شخصي  
- صفحة تسجيل دخول + تسجيل مستخدمين
