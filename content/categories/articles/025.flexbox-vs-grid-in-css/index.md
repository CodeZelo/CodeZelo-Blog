## الجزء الثالث: ما هي خاصية flex-box في CSS؟

الـ Flexbox (أو Flexible Box) صُمم ليكون "مرناً". وظيفته الأساسية هي توزيع المساحة بين العناصر في صف واحد (أفقي) أو عمود واحد (رأسي).

### 1. فلسفة العمل في Flexbox

يعتمد Flexbox على مبدأ "المحور الأساسي" (Main Axis). أنت تضع العناصر داخل حاوية (Container)، وتخبر المتصفح كيف يوزعها.

**أهم خصائص Flexbox:**

* `justify-content`: لتوزيع العناصر أفقياً.
* `align-items`: لمحاذاة العناصر عمودياً.
* `flex-direction`: لتحديد ما إذا كان الاتجاه صفاً أم عموداً.

**مثال كود بسيط:**

```css
.navbar {
  display: flex;
  justify-content: space-between; /* توزيع العناصر على الأطراف */
  align-items: center; /* محاذاة في المنتصف عمودياً */
}

```

## الجزء الرابع: نظام Grid — الشبكة المتقدمة

إذا كان Flexbox هو "خيط" تنظم فيه الخرز، فإن **Grid** هو "ورقة مربعات" كاملة يمكنك وضع أي عنصر في أي مربع فيها.

### 2. لماذا نستخدم Grid؟

لأنه ثنائي الأبعاد (2D). يمكنك التحكم في الصفوف والأعمدة في نفس الوقت وبدقة مذهلة. هو الحل المثالي لهيكل الصفحة كاملاً (Header, Sidebar, Main Content, Footer).

**أهم خصائص Grid:**

* `grid-template-columns`: لتحديد عدد وأحجام الأعمدة.
* `grid-gap`: لتحديد المسافات بين المربعات.

**مثال كود Grid:**

```css
.main-layout {
  display: grid;
  grid-template-columns: 200px 1fr; /* عمود ثابت وعمود مرن */
  gap: 20px;
}

```

## الجزء الخامس: ما هو الفرق بين Flexbox و Grid؟

لنتوقف هنا ونحسم الجدل. **ما الفرق بين CSS grid و flexbox؟** الإجابة تكمن في "البعد".

**جدول المقارنة الحاسمة:**

| وجه المقارنة | Flexbox (البعد الواحد) | Grid (ثنائي الأبعاد) |
| --- | --- | --- |
| **محور العمل** | يعمل على محور واحد (إما صف أو عمود) | يعمل على محورين (صف وعمود معاً) |
| **المنطق** | المحتوى أولاً (Content-first) | الشبكة أولاً (Layout-first) |
| **المحاذاة** | ممتازة للأجزاء الصغيرة والتوزيع الخطي | ممتازة للهياكل الكبرى والتقسيمات المعقدة |
| **التداخل** | العناصر لا تتداخل بسهولة | يمكنك وضع عناصر فوق بعضها في الشبكة بسهولة |

## الجزء السادس: متى يجب إستعمال grid و flex في css؟

هذا هو السؤال العملي الأهم. كقاعدة عامة يتبعها كبار المطورين:

### When to use CSS Flexbox?

* للقوائم العلوية (Navigation Bars).
* لتوسيط عنصر واحد داخل صندوق (Center a div).
* عندما لا تعرف عدد العناصر التي ستظهر (مثلاً قائمة أزرار متغيرة).

### Do people still use CSS grids?

* نعم وبقوة! تستخدم للـ Page Layout بالكامل.
* لمعارض الصور (Image Galleries) التي تتطلب أحجاماً مختلفة.
* لأي تصميم يتطلب محاذاة دقيقة في الطول والعرض معاً.

## الجزء السابع: هل يمكنني استخدام CSS و grid و flexbox معًا؟

الإجابة هي **نعم، وبالتأكيد!** في الواقع، المواقع الاحترافية في 2025 تُبنى بهذه الطريقة. يمكنك استخدام **Grid** لتخطيط الهيكل العام للموقع، ثم استخدام **Flexbox** داخل إحدى خلايا الشبكة لترتيب الأيقونات أو النصوص.

> **اقتباس مميز:** "Grid هو الإطار الخارجي للمنزل، وFlexbox هو ترتيب الأثاث داخل الغرف."

## الجزء الثامن: مستقبل التخطيط في 2025

بحلول عام 2025، لم يعد السؤال **Which is better, CSS Grid or Flexbox?** بل أصبح السؤال كيف نوظفهما مع تقنيات مثل الـ Container Queries.

* **Is Flexbox still used in 2025?** نعم، هو الأداة الأسرع والأخف لتنسيق المكونات الصغيرة (Components).
* **دعم المتصفحات:** أصبح النظامان مدعومين بنسبة 99% من المتصفحات، لذا لا داعي للقلق.

## الجزء التاسع: تطبيقات عملية وملاحظات هامة

* **نصيحة للمحترفين:** استخدم وحدة `1fr` في Grid بدلاً من النسبة المئوية لتجنب مشاكل الحسابات المعقدة.
* **ملاحظة حول التجاوب (Responsiveness):** كلا النظامين يدعمان الـ Media Queries ببراعة، مما يجعل جعل موقعك يعمل على الموبايل أمراً في غاية السهولة.

### الأسئلة الشائعة

> **س: هل تعلم Grid أصعب من Flexbox؟**
> ج: الـ Grid يحتوي على خصائص أكثر، لكن بمجرد فهم منطق "الأعمدة والصفوف" ستجده ممتعاً ومنطقياً جداً.

> **س: هل يؤثر استخدام Grid على سرعة الموقع؟**
> ج: أبداً، المتصفحات الحديثة مصممة لمعالجة هذه الأنظمة بسرعة فائقة، بل هي أفضل أداءً من الطرق القديمة.

**الخلاصة:**
لقد عرفنا اليوم أن **الفرق بين Flexbox و Grid** هو فرق في الوظيفة وليس في الجودة.

1. استخدم **Flexbox** للتخطيطات الخطية والبسيطة.
2. استخدم **Grid** لبناء الهياكل المعقدة والشبكات ثنائية الأبعاد.
3. لا تتردد في دمجهما معاً لتحصل على أفضل ما في العالمين.

**الخطوة القادمة:**
بما أنك أتقنت فنون التخطيط، فأنت الآن جاهز للمرحلة النهائية في CSS: **Media Queries**. سنعرف في الدرس القادم كيف نحول كل هذه المهارات إلى "تصميم متجاوب" يتغير شكله بذكاء حسب حجم الشاشة.

**هل تود أن نقوم بتصميم تخطيط كامل لموقع إلكتروني يستخدم Grid للهيكل وFlexbox للقائمة الآن؟**
