أهلاً بك يا صديقي المبدع في محطتنا الأكثر أهمية وتأثيراً في جمالية المواقع. بعد أن تعلمنا كيف نختار الألوان المناسبة وننسق الخلفيات ونقيس العناصر بالوحدات الصحيحة، حان الوقت لنتعلم كيف نجعل "الرسالة" نفسها جذابة.

**تنسيق النصوص في CSS** هو العلم الذي يحول الكلمات الجامدة إلى تجربة بصرية مريحة وممتعة للزائر.
النصوص هي أكثر ما يتفاعل معه الزوار في أي موقع، وإذا لم تكن واضحة ومنسقة باحترافية، فسيغادر الزائر صفحتك فوراً مهما كانت قيمة المحتوى. في هذا الدليل، سنكتشف "كيف يمكننا تنسيق النص في CSS" لنصل إلى احترافية تضاهي كبرى المواقع العالمية.

## أهمية تنسيق النصوص

تخيل أنك تقرأ رسالة علي الواتساب بخط صغير جداً، أو حروف متلاصقة، أو ألوان غير مريحة؛ بالتأكيد ستشعر بالإرهاق. في الويب، نحن لا نصمم شكلاً فقط، بل نصمم "قابلية القراءة" (Readability).

هذا الدليل المرجعي موجه لكل من:

- مبتدئ يبحث عن **ما هي طريقة تنسيق النصوص باستخدام CSS؟** ليبدأ بداية صحيحة.
- مطور يريد معرفة **كيف يمكنني توسيط النص في CSS** والتحكم في أماكن العناصر بدقة.
- مصمم يسعى لفهم **المسافة بين الاسطر في css** لخلق توازن بصري مريح للعين.

أعدك بنهاية هذا المقال أنك ستتعامل مع النصوص كأنها لوحة فنية، تملك كل أدوات التحكم في حجمها، ولونها، ومكانها، بل وحتى حركتها.

## محتويات دليل تنسيق النصوص الشامل

لقد رتبنا لك هذا الدليل ليغطي كل تفاصيل النص من الألف إلى الياء:

1. **أساسيات الخطوط:** النوع، الحجم، والسمك (Font Properties).
2. **محاذاة وتموضع النص:** كيف تتحكم في "مكان النص في CSS" وتوسيطه.
3. **زخرفة وتجميل النص:** المسافات، الظلال، وحالة الحروف.
4. **تنسيقات الروابط:** جعل اللينكات تفاعلية وجذابة.
5. **تطبيقات متقدمة:** التحريك والظلال.

> **ملاحظة:** تذكر دائماً أن "صفحات التنسيق النمطية css تستخدم للتحكم في مظهر النص المكتوب بلغة html"، بينما الـ HTML وظيفته فقط إظهار النص الخام.

## خصائص الخط الأساسية (Font Properties)

أول ما نفعله هو اختيار "شخصية" الخط وحجمه.

### 1. font-family — نوع الخط

نستخدم هذه الخاصية لنحدد الخط الذي سيراه المستخدم. يفضل دائماً وضع أكثر من خط كـ "خطة بديلة" في حال لم يتوفر الخط الأول على جهاز الزائر.

**مثال احترافي:**

CSS

```css
body {
  font-family: "Cairo", "Arial", sans-serif;
}
```

### 2. font-size —  حجم الخط CSS

لا تستخدم "كود حجم الخط html" القديم؛ البيكسل بل استخدم CSS للتحكم الكامل. يفضل استخدام وحدة **rem** لتكون النصوص متجاوبة مع كافة الشاشات.

```css
h1 {
  font-size: 1rem;
}
```
### 3. font-weight — سمك الخط

هل تريد النص رفيعاً أم عريضاً؟ القيم تتراوح من 100 إلى 900.

```css
h2 {
  font-weight: 700;
}
```
- `400`: عادي (Normal).
- `700`: عريض (Bold).
    
## محاذاة وتحريك النص (Text Alignment)

كثيراً ما نتساءل: "كيف يمكنني توسيط النص في CSS؟" أو كيفية "التحكم في مكان النص CSS".

### 1 text-align — المحاذاة الأفقي

تستخدم هذه الخاصية لتحديد اتجاه النص داخل العنصر16.

- `center`: لتوسيط النص (وهو البديل الاحترافي لـ "كود توسيط النص HTML" القديم).
    
- `justify`: لجعل الأسطر متساوية من الطرفين (مثل الكتب).
    

**جدول قيم محاذاة النص:**

|**القيمة**|**الوظيفة**|**الاستخدام الشائع**|
|---|---|---|
|`center`|توسيط النص في المنتصف|العناوين الرئيسية|
|`right`|محاذاة لليمين|النصوص العربية|
|`left`|محاذاة لليسار|النصوص الإنجليزية|
|`justify`|ضبط السطور من الجانبين|الفقرات الطويلة|

### 2. تحريك النص في CSS

بدلاً من الوسوم القديمة الملغاة مثل `<marquee>`، نستخدم الآن خاصية `animation` في CSS لتحريك النصوص بمرونة وانسيابية عالية. وسنقوم بشرحها في درس مخصص لاحقاً.

## المسافات والزخرفة (Text Decoration)

التفاصيل الصغيرة هي ما تجعل التصميم مريحاً.

### 1. line-height — المسافة بين الاسطر في css

هذه الخاصية هي سر جمال الفقرات الطويلة. القيمة المثالية غالباً ما تكون بين `1.5` و `1.8` لجعل القراءة مريحة.

### 2. text-decoration — زخرفة النص

سؤال مهم: "ما هي القيم التي تأخذها خاصية زخرفة النص في CSS؟".

- `none`: لإزالة الخطوط (تستخدم بكثرة مع الروابط).
- `underline`: لوضع خط تحت النص.
- `line-through`: لشطب النص (كأنه محذوف).

> ملاحظة هامة: "تنسيق نمط الخط في الكود font style italic هو عادي صواب خطأ؟"
> 
> الإجابة هي خطأ؛ لأن italic تجعل الخط مائلاً، بينما القيمة الافتراضية للخط العادي هي normal.

## تنسيق الروابط في css

الروابط (Links) لها طبيعة خاصة لأنها تتفاعل مع المستخدم. المحترفون يغيرون شكل الرابط عند مرور الماوس عليه (Hover).

**مثال لتنسيق الروابط:**
CSS

```css
a {
  text-decoration: none; /* إزالة الخط الافتراضي */
  color: #3498db;
}

a:hover {
  text-decoration: underline; /* إظهار الخط عند مرور الماوس */
  color: #2980b9;
}
```

لقد تعلمنا اليوم كيف نحول صفحة الويب من مجرد نصوص عشوائية إلى واجهة منظمة واحترافية.

**أهم النقاط التي غطيناها:**

1. تغيير نوع الخط وحجمه باستخدام وحدات مرنة مثل **rem**.
2. توسيط النصوص والتحكم في تموضعها بـ `text-align`.
3. تحسين القراءة عبر ضبط **المسافة بين الاسطر في css**.
4. إزالة أو إضافة الزخارف باستخدام `text-decoration`.
5. التعامل مع الروابط والحالات التفاعلية لها.

## الأسئلة الشائعة حول تنسيق النصوص في CSS

> س: هل يؤثر نوع الخط على سرعة الموقع؟
> 
> ج: نعم، يفضل استخدام خطوط النظام الأساسية أو استدعاء خطوط خارجية (مثل Google Fonts) بحذر لعدم إبطاء التحميل.

> س: كيف أجعل النص يظهر بظلال خلفه؟
> 
> ج: نستخدم خاصية text-shadow لتحديد لون وإزاحة الظل، مما يعطي النص عمقاً بصرياً رائعاً.

الخطوة القادمة:

بما أنك أتقنت تنسيق النصوص، فأنت الآن جاهز للمرحلة الكبرى: التخطيط (Layouts). في الدروس القادمة، سنتعلم كيف نرتب هذه النصوص والصور في أعمدة وصفوف لنبني هيكل موقع متكامل.
